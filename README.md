# Agent 学习记录

这个仓库用于持续记录 Agent 学习过程：学习日志、主题笔记、实践项目、资料索引和阶段复盘都放在这里。

## 目录结构

```text
.
├── docs/                # 学习路线、阶段复盘、长期沉淀文档
├── logs/                # 按月份记录每日学习进展
├── notes/               # 按主题整理 Agent 知识笔记
├── projects/            # 小实验、小项目和代码实践
├── resources/           # 文章、课程、论文、工具链接
├── scripts/             # 辅助脚本
└── templates/           # 笔记和复盘模板
```

## 推荐使用方式

1. 每天学习后，在 `logs/YYYY-MM.md` 里补一条记录。
2. 遇到可复用知识点，整理到 `notes/` 下的主题笔记。
3. 做实验或写 Demo 时，放到 `projects/`，并在对应 README 里写清楚目标、运行方式和结论。
4. 每周或每个阶段结束后，在 `docs/` 中写复盘，明确下一阶段要补的短板。

## 快速新建主题笔记

```bash
./scripts/new-note.sh "tool-use"
```

执行后会在 `notes/` 下生成一份基于模板的新笔记。

## 维护原则

- 日志写事实：今天学了什么、做了什么、卡在哪里。
- 笔记写结构：概念、流程、关键代码、踩坑和适用场景。
- 项目写闭环：目标、实现、运行、结果和改进点。
- 资源写判断：为什么值得看、解决了什么问题、读完后沉淀在哪里。

## Git 工作流

```bash
git status
git add .
git commit -m "docs: update agent learning notes"
git push
```

建议每次学习结束后提交一次，提交信息尽量说明本次记录的主题。
