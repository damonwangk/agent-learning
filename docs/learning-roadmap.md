# Agent 学习路线

## 第一阶段：基础概念

- Agent 的定义、边界和典型架构
- LLM、工具调用、记忆、规划、执行循环之间的关系
- ReAct、Plan-and-Execute、Reflection 等常见模式

## 第二阶段：工具调用与工作流

- Function Calling / Tool Use
- 结构化输出与 JSON Schema
- 多步骤任务拆解和错误恢复
- 本地工具、Web 工具、数据库工具的封装方式

## 第三阶段：记忆与知识库

- 短期记忆、长期记忆、会话状态管理
- RAG、向量数据库、检索增强生成
- 知识更新、引用溯源和结果评估

## 第四阶段：多 Agent 协作

- 角色拆分与任务分派
- 协作协议、消息格式和状态同步
- 评审 Agent、执行 Agent、规划 Agent 的组合方式

## 第五阶段：评估与上线

- 成功率、成本、延迟、稳定性评估
- 回归测试集和失败样本库
- 日志、监控、权限、安全边界
